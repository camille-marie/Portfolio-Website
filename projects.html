Surround-3 Engine

Language: C++ Type: Real-time Audio System Complexity: Advanced

A sophisticated real-time granular synthesis system with live audio processing capabilities, built from the ground up in C++. This project represents my journey into low-level audio programming and real-time systems design.

Key Features

  • Real-time granular synthesis with configurable parameters
  • Live audio processing and manipulation
  • Multi-channel audio support
  • Low-latency performance optimization
  • Modular architecture for extensibility

Technical Implementation

The system is built using modern C++17 features and follows audio programming best practices:

// Example of the core synthesis loop
class GranularSynthesizer {
private:
    std::vector<Grain> activeGrains;
    AudioBuffer<float> outputBuffer;
    
public:
    void processAudio(float* input, float* output, int numSamples) {
        // Update grain states
        updateGrains();
        
        // Process each grain
        for (auto& grain : activeGrains) {
            if (grain.isActive()) {
                grain.process(output, numSamples);
            }
        }
        
        // Apply effects and output
        applyEffects(output, numSamples);
    }
};

Pro Tools Session Analyzer

Language: C++ Type: Audio Analysis Tool Complexity: Intermediate

A C++ tool that parses Pro Tools .ptx session files and generates comprehensive analysis reports. Built for EP-381 Audio Engineering coursework to validate project requirements and analyze session structures.

Key Features

  • Binary file parsing of Pro Tools session format
  • Track layout analysis and classification
  • Audio file detection and cataloging
  • Sample rate and configuration detection
  • Automated recommendations and warnings

Usage Example

// Compile and run
g++ -std=c++17 -o protools_analyzer protools_analyzer.cpp
./protools_analyzer "EP381 P2 MyProject.ptx"

// Output includes:
// - Track count and types
// - Audio file references
// - Session configuration
// - Potential issues or warnings
